# Leica Aperio FL 訂貨產品資訊

**注意：研究級多模式機種 (Research Multi-modal Scanner) 配置極度客製化，牽涉激發光源、濾鏡波段、相機及分析軟體之搭配。最終硬體與軟體料號，請務必諮詢您的 Leica 產品專員 (Product Specialist) 進行架構確認。**

---

## 💻 主機型號 (Base Systems)

新世代研究型掃描機提供兩種容量選擇，以滿足不同規模的實驗室：

*   **Aperio FL10 (10 片裝)**
    *   產品定位：適合小型或專注型研究室 (Focused investigations)
    *   進樣方式：自動托盤式 (Tray)
*   **Aperio FL120 (120 片裝)**
    *   產品定位：適合高通量大型研究中心、空間蛋白質體學 (TMA) 篩查或 Core Facility
    *   進樣方式：全自動多盤進樣

---

## ⚙️ 關鍵硬體選配 (Core Configuration Options)

1.  **相機 (Cameras)**:
    *   **Next-Gen Monochrome Fluorescence Camera**：專為微弱螢光訊號優化的次世代超低雜訊單色相機。
    *   **Brightfield Camera (RGB)**：獨立的高保真彩色相機，用於常規 H&E 或 IHC 明視野標定。
    *   *(通常出廠即為雙相機高階配置以發揮 100% 效能)*
2.  **光源 (Light Sources)**:
    *   **X-Cite LED 螢光光源**：配備液態光導管 (Liquid light guide)，確保全視野光照強度均勻，支援精準 LED 強度調節功能。
3.  **物鏡 (Objectives)**:
    *   標準組態通常涵蓋 10x, 20x, 40x 等世界頂級 Leica 光學鏡片。
    *   **63x 自動油鏡 (Auto-oiled Objective)**：高階選配。*(若選配包含油鏡，系統即內含專利之無人值守點油裝置)。*
4.  **濾鏡與電動轉輪 (Filter Turret & Cubes)**:
    *   **8 孔濾鏡轉輪**：可容納至多 7 色螢光探針通道，完美支援極為複雜的 Multiplex IF (多色免疫螢光) 實驗。
    *   需仔細比對客戶實驗所使用之所有螢光染劑波長範圍，挑選並訂購精確匹配之對應濾鏡塊。

## 🧠 終極分析軟體選配 (Indica Labs HALO Ecosystem)

Aperio FL 產品線的最大賣點為**預設搭載/推薦升級**至當代學術界黃金標準定量分析平台：

*   **HALO 核心分析平台**：包含用於腫瘤微環境 (TME) 研究的各類模組，如 Multiplex IF、Spatial Analysis 等。
*   **HALO AI 深度學習擴充**：供研究員訓練專屬神經網絡，進行高精度的組織/細胞分割分類。
*   **HALO Link 影像協作系統**：為 Core Facility 或多中心跨國研究機構設計的安全網頁影像分享、專案權限控管與遠端檢視模組。

## 📝 訂購前關鍵訪談清單 (Qualification Checklist)

為確保報價精確且無遺漏，請於報價前與研究員確認下列五大問題：

1.  **「您的實驗最多會同時標記到幾種螢光色？染劑光譜為何？」**
    *(決定 8 孔濾鏡轉輪內的濾光片型號與配置)*
2.  **「是否需要觀察次細胞結構 (Subcellular) 或極微小 FISH 訊號點？」**
    *(決定是否必須配備 63x 自動油鏡)*
3.  **「掃描的玻片厚度為何？是否包含神經腦組織等極厚切片？」**
    *(決定焦點堆疊 Z-Stacking 功能之需求及資料存儲評估)*
4.  **「每週預估掃描量為何？是否包含數百點之大型組織微陣列 (TMA)？」**
    *(決定推廣 FL10 或 FL120)*
5.  **「產出影像後，您打算如何分析『細胞之間的空間鄰近距離』？」**
    *(以此痛點切入推廣高單價的 HALO / HALO AI 軟體套件)*
