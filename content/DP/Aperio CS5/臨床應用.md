> 本文件說明 Aperio CS5 在臨床上的實際使用場景，協助業務明確向客戶說明「哪些情境最適合這台機器」。

| 科室 | 應用場景 |

## 🔪 臨床應用一：術中冰凍切片遠距診斷 (Intraoperative Frozen Section)

### 應用情境
外科手術進行中，需要病理醫師即時判斷：
- **切緣 (Margin)** 是否已達到無瘤狀態
- **淋巴結 (Lymph Node)** 有無轉移
- **腫瘤性質** 確認（良性 vs. 惡性）

### 傳統流程痛點
- 病理醫師必須**人在現場**或在手術樓旁的顯微鏡旁等待
- 白天一位醫師可能需要同時應付多個手術室的冰凍請求

### CS5 的解決方案
1. CS5 放置於**冰凍切片機旁**（體積小可放桌上）
2. 技術員切片染色後直接放入 CS5，**<1 分鐘**完成掃描
3. 病理醫師透過任何電腦/平板上**遠端瀏覽影像**並回報診斷
4. 手術醫師即時得知結果，縮短病人在手術台上的等待時間

### CS5 為什麼適合這個場景
- **掃描速度快**：20x 僅需約 50 秒，急件不等人
- **5 片載量**：一次放入多個急件同步排程，效率高於單片機
- **體積小巧**：能放在手術室外走廊或冰凍切片室桌上

## 📋 臨床應用二：大玻片 (2×3 Inch Macro Slide) 掃描

### 何謂大玻片？
標準組織切片玻片尺寸為 1×3 英吋。但以下檢體需要使用 **2×3 英吋大玻片**：
- **乳房根治切除術 (Mastectomy)**：腫瘤面積大，標準片放不下
- **前列腺根治 (Radical Prostatectomy)**：全切面病理評估
- **腦腫瘤 (Brain Tumor)**：大塊腦組織切面
- **皮膚腫瘤廣泛切除**：需要完整呈現切緣

### 臨床意義
大玻片確保病理醫師能在**一張影像**中看到完整的腫瘤邊界、切緣距離與周邊組織關係，對腫瘤分期與切緣判斷至關重要。

### CS5 的優勢
**CS5 是 Leica Aperio 家族中唯一支援 2×3 英吋大玻片掃描的機型**，配合選配的大玻片載具即可使用，無需額外購買大型掃描機。

## 🌐 臨床應用三：偏鄉醫院數位病理導入

### 應用情境
- 縣市立醫院或地區醫院，每日玻片量不高（< 30 片）
- 全職病理醫師難以招募，需要遠距會診支援
- 有意開始數位病理轉型，預算有限

### CS5 為什麼是入門首選
| 優勢 | 說明 |
| :--- | :--- |
| **低建置成本** | 售價相對 GT 系列親民，適合預算有限的中小型醫院 |
| **空間需求小** | 桌上型，不需要獨立機房 |
| **功能完整** | Z-Stack、手動掃描、診斷級影像，功能不因售價打折 |
| **擴充路徑清晰** | 日後業務量增加，可升級至 GT 180/GT 450，影像管理系統 (eSlide Manager) 完全相容 |

| 場景 | 適合 CS5？ | 說明 |
