# CytoInsight GSL 臨床應用

## 主要臨床應用場景

### 1. 血液腫瘤 FISH 檢測（最核心市場）

血液腫瘤是 CytoInsight GSL 最重要的臨床應用領域，常見檢測項目包括：

| 疾病 | FISH 探針 | 臨床意義 |
| :--- | :--- | :--- |
| 慢性骨髓性白血病 (CML) | BCR-ABL1 | 診斷與治療監測（伊馬替尼療效） |
| 急性淋巴性白血病 (ALL) | BCR-ABL1、MLL | 預後分層 |
| 多發性骨髓瘤 (MM) | del(17p)、t(4;14)、t(14;16) | 高風險分層、治療選擇 |
| 慢性淋巴性白血病 (CLL) | del(13q)、del(11q)、del(17p)、+12 | CLL 預後四組分類 |
| 瀰漫性大B細胞淋巴瘤 (DLBCL) | MYC、BCL2、BCL6 | Double/Triple Hit 淋巴瘤診斷 |

### 2. 組織 FISH 檢測（固態腫瘤）

| 癌症類型 | FISH 項目 | 臨床意義 |
| :--- | :--- | :--- |
| 乳癌 | HER2 基因擴增 | 決定是否使用 Herceptin（曲妥珠單抗） |
| 非小細胞肺癌 (NSCLC) | ALK、ROS1 重排 | 決定是否使用 ALK 抑制劑（如 crizotinib） |
| 非小細胞肺癌 (NSCLC) | MET 擴增 | 靶向治療選擇 |
| 胃癌 | HER2 基因擴增 | 指導 Herceptin 使用 |
| 膀胱癌 | UroVysion (FISH) | 早期膀胱癌偵測 |

### 3. 染色體微缺失與重排

*   **神經母細胞瘤**：MYCN 擴增分析（預後關鍵指標）
*   **葡萄膜黑色素瘤**：Monosomy 3 分析
*   **軟組織肉瘤**：SS18-SSX（滑膜肉瘤）、EWSR1（尤文肉瘤）重排

## GSL10 vs GSL120 適用情境

### CytoInsight GSL10（適合中小型實驗室）

**典型客戶**：
*   地區醫院病理科（日均 FISH 量 < 20 件）
*   以血液科 FISH 為主的實驗室

**優勢話術**：
> 「GSL10 讓貴院不需要承擔大型平台的預算壓力，就能享有自動化掃描的效率提升與報告品質標準化。」

### CytoInsight GSL120（適合大型醫學中心）

**典型客戶**：
*   醫學中心病理科（日均 FISH 量 > 50 件）
*   同時承接血液 FISH 與組織 FISH 的綜合平台

**優勢話術**：
> 「GSL120 的 120 片容量讓夜間無人值守的批次掃描成為可能，隔天早上技術員來就能直接進入判讀流程，大幅提升科室產能。」

## 工作流程整合

```
樣本製作（探針雜交/沖洗）
        ↓
CytoInsight GSL 自動掃描（GSL10 或 GSL120）
        ↓
CytoVision 工作站影像管理與初步分析
        ↓
病理醫師 / 細胞遺傳學師 判讀
        ↓
出具 IVD 臨床報告
```

## 與傳統人工顯微鏡的比較優勢

| 比較項目 | 傳統人工判讀 | CytoInsight GSL |
| :--- | :--- | :--- |
| **效率** | 每件需 30–60 分鐘 | 自動掃描，批次處理 |
| **再現性** | 依賴技術員個人經驗 | 標準化掃描參數，一致性高 |
| **樣本保存** | 必須當下判讀，燈光漂白 | 數位影像永久保存，可重複閱覽 |
| **遠端會診** | 需郵寄玻片 | 直接數位傳送，即時遠端判讀 |
| **報告效率** | 手動記錄訊號數 | 半自動計數輔助，加速報告 |
