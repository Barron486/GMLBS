# Aperio HALO AP 臨床應用

> 本文件說明 Aperio HALO AP 在醫院臨床病理工作流中的應用場景。HALO AP 不是掃描機，而是讓「掃描後的影像真正發揮臨床價值」的影像管理與 AI 分析平台。

---

## 🏥 主要臨床科室

| 科室 | 應用場景 |
| :--- | :--- |
| **病理科/解剖病理科** | 數位玻片影像管理、遠距閱片、多科 MDT 協作 |
| **分子病理室** | AI 輔助 IHC 定量（HER2、PD-L1、Ki-67）、FISH 影像管理 |
| **血液腫瘤科病理** | AI 骨髓切片分析、淋巴瘤分類輔助 |
| **研究與轉譯醫學中心** | 搭配 HALO AI 進行腫瘤免疫微環境等研究型分析 |

## 📋 臨床應用一：數位病理初判 (Primary Diagnosis) — FDA 510(k) 認證

### 什麼是「初判」？
病理醫師在**電腦螢幕上**直接進行第一次診斷，完全不需要回到光學顯微鏡確認。

### 法規認證組合
只有通過美國 FDA 510(k) 認證的「掃描機 + 影像管理系統」組合，才能合法用於臨床初判：
- **認證組合**：Aperio GT 450 DX（掃描機）+ Aperio HALO AP（閱片平台）

### 對醫院的意義
- 病理醫師無需人在現場，可在辦公室、在家、或異地閱片
- 廢除「必須人在顯微鏡旁才能發報告」的限制
- 真正實現**數位病理全流程自動化**

## 🤖 臨床應用二：AI 輔助 IHC 定量

### 常見 AI 評分項目

| IHC 標記 | 適用癌症 | AI 輸出 | 臨床決策意義 |
| :--- | :--- | :--- | :--- |
| **HER2** | 乳癌、胃癌 | 0/1+/2+/3+ 自動分類 | 決定 Trastuzumab、T-DM1 使用 |
| **PD-L1 TPS** | 非小細胞肺癌 | 腫瘤細胞陽性百分比 | 決定 Pembrolizumab 一線/二線 |
| **PD-L1 CPS** | 胃癌、頭頸癌 | 合計正計分 | 決定免疫治療適應症 |
| **Ki-67** | NET、乳癌 | 陽性細胞百分比 | 腫瘤分級 G1/G2/G3 |
| **ER/PR Allred** | 乳癌 | 陽性比例 × 強度 | 決定荷爾蒙治療適用性 |

> 💡 **零額外步驟**：AI 在掃描後自動背景執行，醫師打開案例時結果已就緒。

## 🌐 臨床應用三：遠距病理會診

| 場景 | 說明 |
| :--- | :--- |
| **次專科支援** | 偏鄉醫院缺乏神經/皮膚病理醫師 → 外院顧問透過瀏覽器遠距閱片 |
| **術中急件** | 病理醫師從家中登入 HALO AP 即時判讀冰凍切片 |
| **多院區集中判讀** | 各分院掃描後，醫師在自己辦公室透過瀏覽器簽核報告 |
| **MDT 腫瘤討論** | 腫瘤科、外科、放射科在同一螢幕共同討論病理影像 |

## 🧪 臨床應用四：臨床試驗影像管理
