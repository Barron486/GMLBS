# Aperio HALO AP — AI 驅動臨床數位病理影像管理平台

## 🌟 產品定位與一句話賣點

> 「Aperio HALO AP 是 Leica Biosystems 與 Indica Labs 聯手打造的**臨床等級數位病理影像管理平台**，已取得美國 FDA 510(k) 認證。它不只是影像瀏覽器，更是將掃描儀影像轉化為**可追溯、可分析、可協作**的臨床與研究資產的核心中樞。」

---

## 📖 產品起源與背景

### Indica Labs × Leica Biosystems 的策略聯盟

*   **Indica Labs** 成立於美國新墨西哥州，是全球數位病理影像分析軟體的領導廠商，以旗下研究級產品 **HALO** 與 **HALO AI** 聞名學術界。
*   2020 年起，Leica Biosystems 與 Indica Labs 建立策略合作關係，共同開發專為**臨床工作流**優化的影像管理平台。
*   **Aperio HALO AP** 是這個合作的成果——它繼承了 Indica Labs 在 AI 影像分析的深厚底蘊，同時由 Leica 的臨床工作流經驗加以重新打造，確保符合醫療法規（FDA 510(k)、CE-IVDR）與醫院 IT 標準。
*   **命名釐清**：
    *   **HALO**（研究版）：Indica Labs 直銷，面向學術與藥廠研究。
    *   **HALO AI**（研究版）：HALO 的深度學習擴充包，訓練自訂 AI 模型。
    *   **Aperio HALO AP**（臨床版）：由 Leica Biosystems 獨家銷售，專為醫院病理科設計，搭配 Aperio 掃描儀使用。

### 系統串接能力

| 串接對象 | 協定/方式 | 說明 |
|:---|:---|:---|
| **LIS / LIMS** | HL7 v2.x, REST API | 雙向病例資料同步——從 LIS 接收病例 → 閱片完成後將結果回寫 LIS |
| **PACS / VNA** | DICOM 標準 | 將 WSI 影像以 DICOM WSI 格式歸檔至醫院影像中心 |
| **EMR / HIS** | 開放式 API | 支援在電子病歷中嵌入 HALO AP 影像連結，醫師點選即可閱片 |
| **SSO / AD** | LDAP / SAML | 整合醫院統一身分驗證系統，無需額外帳號管理 |

| 項目 | 支援範圍 |

## 🤖 AI 特色與 Aperio AI Store

### Aperio AI Store 概念

Aperio AI Store 是內建於 HALO AP 的**演算法市場**，病理醫師可直接從中選用經過臨床驗證的 AI 工具，無需自行開發或訓練模型。

### 目前可用的 AI 應用方向

| AI 應用 | 功能說明 | 臨床價值 |
|:---|:---|:---|
| **H&E 組織分類** | 自動辨識腫瘤區域、壞死、淋巴球浸潤等組織型態 | 標準化判讀，減少觀察者間差異 |
| **IHC 定量評分** | PD-L1 TPS/CPS、HER2 (0-3+)、Ki-67 增殖指數自動計算 | 精準醫療伴隨式診斷的客觀輔助 |
| **細胞計數** | 有絲分裂細胞 (Mitotic figures)、免疫細胞等自動計數 | 大幅節省人工計數時間 |
| **品質控制 (QC)** | 自動偵測模糊區域、氣泡、摺痕等掃描瑕疵 | 在病理醫師閱片前預先篩除不合格影像 |

### AI 工作流程

```
玻片掃描完成 → 影像自動進入 HALO AP
                    │
                    ▼
            ┌───────────────┐
            │ AI 自動分析    │ ← 預先設定的 AI 規則
            │ (背景執行)     │    (例：所有 IHC 玻片自動跑 PD-L1 評分)
            └───────┬───────┘
                    │
                    ▼
            ┌───────────────┐
            │ 結果嵌入案例   │ ← 病理醫師開啟案例時，AI 結果已準備好
            │ (AI Overlay)   │    可疊加顯示或隱藏
            └───────────────┘
```

> **💡 關鍵賣點**：AI 是「**背景自動執行**」的。病理醫師不需要手動啟動分析——掃描完成後，預設的 AI 演算法會自動在背景運算，醫師打開案例時結果已經在那裡了。這是真正的「零額外步驟」AI 輔助。

## 🎯 核心應用場景

### 1. 臨床初判 (Primary Diagnosis) — FDA 認證

搭配 GT 450 DX 使用時，HALO AP 是美國 FDA 510(k) 認證的臨床初判平台。病理醫師可合法地在螢幕上進行首次診斷，不需要回到顯微鏡確認。

### 2. 遠距會診 (Teleconsultation)

*   資深病理醫師可在家中或異院透過瀏覽器連入 HALO AP 閱片。
*   適用於：偏鄉醫院缺乏次專科病理醫師、術中急凍需要即時會診、跨院 MDT（多專科團隊會議）。

### 3. 教學與品質保證 (QA)

*   建立教學案例庫，供住院醫師訓練與測驗。
*   自動化品質追蹤：染色品質、掃描品質、判讀一致性等 KPI 統計。

### 4. 臨床試驗影像管理 (Clinical Trial Image Management)

*   藥廠多中心臨床試驗需要集中管理來自各醫學中心的數位玻片影像。
*   HALO AP 提供安全的雲端或地端部署，符合 21 CFR Part 11 電子簽名法規。

1.  **賣「大腦」而非「眼睛」**：「主任，掃描儀只是幫您把玻片拍成照片。真正讓數位病理發揮價值的是 HALO AP——它是您的影像管理中樞、AI 分析引擎、和遠端閱片平台，三合一。」

2.  **強調 FDA 認證的臨床合法性**：「HALO AP 搭配 GT 450 DX 是通過美國 FDA 510(k) 認證的組合。這代表您的病理醫師可以合法地在螢幕上做臨床初判，不需要再回頭看顯微鏡確認——這是真正的數位轉型，不是只是把玻片掃成圖檔。」

3.  **降低導入門檻**：「HALO AP 是瀏覽器介面，技術員和醫師不需要安裝任何軟體。只要有網路和一台電腦，打開 Chrome 就能開始閱片。而且它跟您現有的 LIS 可以雙向串接，不會打亂現有流程。」

4.  **與研究版 HALO 的區隔**：「如果貴院的研究團隊已經在用 Indica Labs 的 HALO，那太好了——HALO AP 的操作邏輯完全相容，研究員幾乎零學習曲線就能上手臨床端。而且未來如果需要開發自訂 AI 模型，我們還有 HALO AI 可以擴充。」
