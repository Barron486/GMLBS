## 🧬 臨床應用一：染色體核型分析 (Karyotyping)

### 適用檢體類型
- 羊水 (Amniotic Fluid)
- 臍帶血 (Cord Blood)
- 絨毛膜採樣 (CVS)
- 周邊血液 (Peripheral Blood)
- 骨髓 (Bone Marrow)

### 臨床意義與診斷目的

| 應用 | 說明 |
| :--- | :--- |
| **產前診斷** | 偵測胎兒染色體數目異常（如唐氏症 Trisomy 21、愛德華氏症 Trisomy 18）或結構異常（倒位、易位） |
| **血液惡性腫瘤** | 偵測白血病、淋巴瘤的染色體異常（如 CML 的 Philadelphia 染色體 t(9;22)），用於診斷分型與預後評估 |
| **不孕症評估** | 分析夫妻雙方染色體結構，排除習慣性流產的染色體因素 |
| **新生兒異常** | 確認先天發育異常嬰兒的染色體病因 |

### CytoInsight GSL 如何幫助
- **AI-Powered Karyotyping**：自動完成染色體切割、配對、排盤，減少 93.6% 手動操作時間
- **Auto Metaphase Finding**：系統自動全片掃描，找出最優質的分裂相，不再需要技術員長時間盯著顯微鏡

## 🔬 臨床應用二：FISH 螢光原位雜交 (Fluorescence In Situ Hybridization)

### 適用檢體類型
- **細胞 FISH**：血液、骨髓、羊水塗片
- **組織 FISH**：石蠟包埋組織切片 (FFPE)

### 臨床意義與診斷目的

| 癌症類型 | 常用 FISH 探針 | 臨床用途 |
| :--- | :--- | :--- |
| **乳癌** | HER2/CEP17 | 確認 HER2 基因擴增，決定是否使用標靶治療（如 Herceptin） |
| **肺癌** | ALK、ROS1、RET | 伴隨式診斷，選擇對應的酪胺酸激酶抑制劑 |
| **膀胱癌** | UroVysion (3、7、17、9p21) | 尿液細胞學 FISH 輔助診斷膀胱癌復發 |
| **白血病/淋巴瘤** | BCR/ABL1、MYC、IGH、BCL2 | 分型診斷、預後分層、追蹤治療反應 |
| **神經母細胞瘤** | MYCN | 決定高低風險分組，指導化療強度 |
| **多發性骨髓瘤** | del(17p)、t(4;14)、t(14;16) | 高風險基因分層 |

### CytoInsight GSL 如何幫助
- **最高 7 螢光通道**：支援多探針同步分析，一次掃描取得完整基因資訊
- **Z-Stack 多焦面融合**：解決組織 FISH 焦平面問題，避免因組織厚度遺漏訊號
- **自動訊號計數**：批次定量分析，消除人眼計數疲勞與主觀誤差
- **智慧 Assay Builder**：設定標準化計數規則，確保不同技術員結果一致

## 🔭 臨床應用三：M-FISH / 多色螢光核型分析

### 適用情境
- 複雜染色體結構異常（多重易位、衍生染色體）
- 常規 G-band 核型分析無法明確辨識的異常
- 實體腫瘤的複雜核型

### 臨床意義
用24色螢光探針標記每條染色體，讓複雜的染色體重組一目了然，精確界定易位的來源染色體——這對血液腫瘤病人的精準預後分層至關重要。

## 🏗️ 臨床應用四：多站點實驗室網絡 (Multi-site Network)

### 適用場景
- **醫學中心帶動衛星醫院**：中心配備 GSL 120 高通量掃描，衛星院所加裝 Review Station，統一由中心細胞遺傳主任發報告
- **國家/省級遺傳檢驗中心**：集中處理來自全省各縣市轉介的染色體病例
- **遠距細胞遺傳會診**：資深醫師在家中或其他醫院，透過內建遠端存取功能審閱影像並發報告

### CytoInsight GSL 如何幫助
- **Review Station**：純軟體授權，安裝在醫師辦公電腦即可遠端閱覽、修改排盤、電子簽核報告
- **Client-Server 架構**：SQL Server 中央資料庫統一管理所有影像與病歷資料
- **Window 11 + 內建遠端存取**：無需額外 VPN 軟體，符合現代醫療資安要求
