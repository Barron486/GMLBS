# Leica Aperio CS5 訂貨產品資訊

**注意：詳細配置與料號請務必使用 Leica 官方報價系統 (SAP/Salesforce) 確認。CS5 為較新機型，料號可能會有變動。**

## 主機型號

*   **Aperio CS5 Scanner (Research)**
    *   Catalog Number: 請洽詢業務 (前代 CS2 為 23CS100 等系列)。
*   **Aperio CS5 DX Scanner (IVD)**
    *   Catalog Number: 請洽詢業務。
    *   描述：符合 IVD 規範的臨床版本。

## 玻片匣 (Magazines)

*   **5-slide Magazine**: 
    *   用於標準 1x3 inch 玻片。
    *   建議多買 1-2 個備用。
*   **2-slide Magazine (Large)**:
    *   用於 2x3 inch 大玻片 (選配)。

## 工作站電腦 (Workstation)

*   CS5 不像 GT 180 內建電腦，通常需要外接一台高性能 PC。
*   規格需求：
    *   CPU: Intel Core i7 或更高。
    *   RAM: 16GB 或 32GB。
    *   Storage: SSD 建議 (用於緩存)。
    *   Monitor: 高解析度專業螢幕 (Barco 或 Eizo 尤佳)。

## 軟體

*   **Aperio Image Analysis (Workstation License)**: 若需進行簡單分析。
*   **eSlide Manager (Hub)**: 若此機作為衛星端點 (Spoke)，需確認是否能連線至總院的 eSlide Manager Hub。

## 訂購注意事項

1.  **電腦規格**：務必確認客戶自備電腦是否符合 Leica 規格，或直接報價含電腦的套組 (Bundle)。
2.  **大玻片需求**：確認客戶是否有大組織切片 (Large format) 需求，若有需加購 2x3 玻片匣及授權。
