# HistoCore PEARL - 桌上型單缸組織處理機

## 產品定位
HistoCore PEARL 是 Leica Biosystems 專為**空間有限、中低通量或預算導向的實驗室**打造的**入門 / 經濟型封閉式組織處理機**。它以極小的佔地面積卻提供 200 盒的處理容量，搭配**最高等級的三重廢氣防護**系統，是取代傳統開放式旋轉脫水機的最佳升級選擇。

## 核心賣點 (銷售話術重點)

### 🏆 賣點一：超小體積、超大容量
*   **機身尺寸**：僅 600 x 710 x 1320 mm（寬 x 深 x 高），可輕鬆安裝在標準實驗桌旁或狹窄角落。
*   **處理容量**：單缸最多 **200 個標準包埋盒** (2 個提籃 x 100 盒)。
*   **銷售話術**：「PEARL 佔地不到一張辦公椅的面積，卻能一次處理 200 盒——這在桌上型等級的機器中幾乎是最大容量。」

### 🛡️ 賣點二：三重廢氣防護 – 人員安全至上
PEARL 提供業界最完整的**三層防護**，將有毒試劑蒸氣暴露降到最低：
1.  **處理缸端直接抽氣 (Extraction at Retort)**：從源頭就抽走揮發性氣體。
2.  **外接排風管介面 (External Exhaust Adapter)**：可接入實驗室中央排風系統。
3.  **活性碳濾網 (Activated Carbon Filter)**：最後一道過濾屏障。
*   **銷售話術**：「三道防線——缸口就先抽走、管路再排出去、最後活性碳再過濾一次。PEARL 是您能找到最安全的桌上型組織處理機。」

### 📋 賣點三：預載驗證程式 + RMS 試劑管理
*   **三組預載驗證程式**：Biopsy (活檢快速)、Standard Overnight (標準過夜)、Cleaning (自動清洗)。
*   **RMS 試劑管理系統**：自動追蹤每瓶試劑的使用量與使用次數，並在需要更換時主動提醒。
*   **USB 紀錄匯出**：所有處理日誌與試劑紀錄可透過 USB 匯出，便於品質管理。
*   **銷售話術**：「PEARL 雖是入門機型卻不馬虎——預載驗證程式讓您拆箱即用，RMS 管理讓您不用擔心忘記換藥。」

### 🎯 賣點四：全封閉式設計 + 防潑安全
*   **全封閉式處理環境**：整個脫水過程在密閉空間中完成，試劑蒸氣不外洩。
*   **抽屜式石蠟浴 (Drawer-Style Paraffin Baths)**：石蠟更換只需拉出抽屜，安全便利。
*   **防潑盤 (Spill Collection Tray)**：石蠟浴與試劑瓶下方均設有防潑盤，完整收集意外溢液。
*   **耐二甲苯觸控螢幕**：專用面板不怕化學藥劑侵蝕。

## 技術規格摘要

| 項目 | 規格 |
|---|---|
| 處理缸數量 | 1 (單缸密閉式) |
| 最大處理容量 | 200 標準包埋盒 (2 提籃 x 100) |
| 石蠟浴 | 3 個 (每浴 3.5L) |
| 試劑瓶 | 11 瓶 + 1 凝結水瓶 + 2 清洗液瓶 (每瓶 3.5L) |
| 石蠟溫度 | 50°C – 70°C |
| 處理缸溫度 | 環境溫度 或 35°C – 60°C |
| 預載驗證程式 | Biopsy / Standard Overnight / Cleaning |
| 試劑管理 | RMS 自動追蹤系統 |
| 廢氣防護 | **三重防護** (缸端抽氣 + 外接排風 + 活性碳) |
| 操作介面 | 耐二甲苯觸控螢幕 (多國語言) |
| 紀錄匯出 | USB 下載 |
| 機身尺寸 | 600 x 710 x 1320 mm (寬 x 深 x 高) |
| 重量 | ≤ 140 kg |

## 適用對象
*   **中小型醫院或診所病理室**：每日處理量 ≤ 100 件。
*   **空間極度受限的實驗室**：需要極小佔地面積的機型。
*   **預算導向的採購案**：以經濟合理的價格升級至全封閉式處理機。
*   **取代傳統開放式旋轉脫水機**：告別有毒蒸氣暴露的舊型設備。
