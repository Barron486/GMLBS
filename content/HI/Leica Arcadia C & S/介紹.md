# Leica Arcadia C & S — 組織包埋中心 (Embedding Center)

## 🌟 產品定位與一句話賣點

> 「Arcadia 是 Leica Biosystems 專為高效、舒適的石蠟組織包埋流程設計的模組化工作站——**Arcadia H** 負責加熱包埋，**Arcadia C** 負責冷卻固化，**Arcadia S** 提供額外的冷凍儲存空間。三者組合即是完整的 Embedding Center。」

---

## 📖 產品系列說明

### 命名釐清

| 型號 | 全名 | 功能 |
|:---|:---|:---|
| **Arcadia H** | Arcadia Heating module | **加熱包埋模組**：提供恆溫石蠟槽、加熱面板與鑷子加熱器，供技術人員將組織放入蠟塊模具中。 |
| **Arcadia C** | Arcadia Cooling module | **冷卻模組（冷盤）**：快速冷卻蠟塊使其凝固，以便後續切片。內建獨立冷凍板與冷凍組織儲存區。 |
| **Arcadia S** | Arcadia Storage module | **冷凍儲存模組**：額外的低溫儲存空間，可暫存已包埋但尚未切片的蠟塊，維持蠟塊在最佳溫度。 |

> **💡 業務須知**：客戶所說的「Arcadia C & S」通常是指冷卻模組 + 儲存模組的組合。但完整的 Embedding Center 需要同時搭配 **Arcadia H（加熱端）**。在報價時務必確認客戶是否已有加熱端，或需要整套 H + C + S。

## 🔗 在 Leica 工作流中的定位

```
PELORIS 3 / PEGASUS / PEARL        Arcadia H + C + S           AUTOCUT / MULTICUT
   (組織脫水處理)          →      (石蠟包埋 & 冷卻)     →      (石蠟切片)
                                        │
                                    蠟塊凝固後
                                    可暫存於 Arcadia S
                                    等待批次切片
```

*   **上游**：PELORIS 3 等組織處理機完成脫水浸蠟後，技術員取出組織放入包埋模具。
*   **Arcadia H**：提供恆溫蠟槽與加熱面板，讓技術員將組織精確定位於蠟塊模具中。
*   **Arcadia C**：蠟塊放上冷卻板快速凝固，凝固速度與品質直接影響後續切片的平整度。
*   **Arcadia S**：高通量實驗室可將大批蠟塊暫存於低溫抽屜中，白班批次切片。
*   **下游**：凝固的蠟塊取出後送至 AUTOCUT / MULTICUT 進行石蠟薄切。

## 🎯 業務銷售重點

### 1. 人體工學 — 減少職業傷害
*   包埋是病理實驗室中**最耗費雙手與肩頸**的工作之一，技術員每天可能需要包埋 200–400 個蠟塊。
*   Arcadia H 的可調高度設計讓技術員可以坐著或站著工作，大面積加熱面板減少手腕翻轉動作。
*   話術：*「包埋工作最怕的是肩頸痠痛和腕隧道症候群。Arcadia 的高度可調與大面板設計，就是為了保護您的技術員團隊的長期健康。」*

### 2. 精確控溫 — 確保切片品質
*   石蠟溫度太高會造成組織損傷，太低會提前凝固導致定位困難。Arcadia H 的 ±1°C 精確控溫確保石蠟始終在最佳狀態。
*   Arcadia C 的快速冷卻能力確保蠟塊均勻凝固——蠟塊凝固不均會導致後續切片出現裂痕或捲曲。
*   話術：*「切片品質的好壞，有一半取決於包埋的品質。Arcadia 的精確溫控從源頭確保每一個蠟塊都是完美的。」*

### 3. 模組化設計 — 彈性擴充
*   小型實驗室可先買 Arcadia H + C 基本組合。
*   通量增加後，加購 Arcadia S 擴充冷凍儲存空間即可，無需更換設備。
*   話術：*「先從 H + C 開始，未來檢體量增加了再加 S 模組就好——不用一次到位，也不用擔心將來要整組換掉。」*

## 📝 常見 FAQ

**Q：客戶說「我只要冷盤就好」，需要報 Arcadia C 嗎？**
A：是的。確認客戶是否已有其他品牌的加熱包埋台。如果有，可以單獨搭配 Arcadia C 做冷卻。但如果客戶要汰換整套，務必報 H + C（+ S 依通量）。

**Q：Arcadia 可以搭配非 Leica 的組織處理機使用嗎？**
A：可以。Arcadia 是獨立工作站，不依賴特定組織處理機的輸出。但在銷售話術上，建議強調「整套 Leica 從 PELORIS → Arcadia → AUTOCUT 的流程優勢」。

**Q：Arcadia 有連網/資料追溯功能嗎？**
A：基礎版 Arcadia 主要是獨立操作的機電設備，無內建數位連網功能。如果客戶對全流程數位追溯有強烈需求，建議搭配 Leica 的 Resolve (Remote Service) 平台或第三方 LIMS 條碼追蹤方案。
