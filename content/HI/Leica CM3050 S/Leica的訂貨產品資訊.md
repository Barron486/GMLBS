# Leica CM3050 S 訂貨資訊

**注意：詳細配置與料號請務必使用 Leica 官方報價系統確認。**

## 主機配置

*   **Leica CM3050 S 標準型**
    *   包含：主機、電動馬達驅動、腔體冷卻系統、快速冷凍架、防靜電切片廢棄物盤。

## 主要選配

*   **獨立樣品頭冷卻系統**：將樣品頭溫度獨立控制至 -50°C (強烈建議)。
*   **電動液壓高度調節**：坐/站姿切換，適應不同操作者身高。
*   **CryoJane Tape-Transfer System**：困難樣品 (脂肪、骨) 的無皺切片解決方案。

## 耗材與配件

| 項目 | 說明 |
|---|---|
| 低剖面刀片 (LP Blade) | 適合精密薄切 |
| 高剖面刀片 (HP Blade) | 適合一般切片 |
| 防靜電切片廢棄物盤 | 保持工作區整潔 |
| 冷凍切片用安全手套 | 保護操作人員 |
| OCT 包埋劑 | 冷凍包埋用 |

## 訂購注意事項
1.  **確認研究需求**：是否需要獨立樣品頭冷卻？是否需要 CryoJane？
2.  **電源規格**：確認實驗室電壓 (110V/220V)。
3.  **安裝空間**：機身約 882 x 766 mm，重約 180 kg，需確認地板承重。
